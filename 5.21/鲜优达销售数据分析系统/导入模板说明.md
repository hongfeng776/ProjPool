# 销售数据导入模板说明

## Excel 文件格式要求

### 支持的格式
- .xlsx (Excel 2007+)
- .xls (Excel 97-2003)

### 文件大小限制
- 最大 10MB

---

## 列说明

| 列名 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 商品名称 | ✅ 是 | 必须与系统中已有的商品名称完全一致 | 大白菜、苹果、猪肉 |
| 销售数量 | ✅ 是 | 数字，必须大于0 | 10.5、5、3.2 |
| 销售金额 | ✅ 是 | 数字，必须大于0 | 36.75、128.00 |
| 销售日期 | ✅ 是 | 日期格式，YYYY-MM-DD | 2026-05-20 |
| 客户类型 | ⭕ 否 | 默认为"散客" | 散客、会员、批发 |

## 数据校验规则

### 1. 必填项校验
- 商品名称、销售数量、销售金额、销售日期不能为空

### 2. 商品验证
- 商品名称必须与系统中已存在的商品完全匹配
- 不匹配的商品会导致导入失败

### 3. 数值验证
- 销售数量必须大于 0
- 销售金额必须大于 0
- 数值格式不正确会导致导入失败

### 4. 日期验证
- 销售日期格式必须正确（YYYY-MM-DD）
- 支持 Excel 日期格式自动识别

### 5. 重复数据检测
- 完全相同的记录（商品ID、数量、金额、日期都相同）会自动跳过
- 跳过的记录会在导入结果中显示

## 导入结果说明

导入完成后会显示以下统计信息：

| 统计项 | 说明 |
|--------|------|
| 总记录数 | Excel 文件中的数据总行数 |
| 成功导入 | 成功添加到数据库的记录数 |
| 重复跳过 | 因重复而被跳过的记录数 |
| 导入失败 | 验证失败或异常的记录数 |

## 常见问题

### Q: 导入失败提示"商品不存在"怎么办？
A: 请检查商品名称是否与系统中完全一致，包括空格和特殊字符。可以在"商品管理"页面查看现有商品列表。

### Q: 哪些情况会被判定为重复数据？
A: 当某条记录的 **商品ID、销售数量、销售金额、销售日期** 四个字段与数据库中已有记录完全相同时，会被判定为重复数据。

### Q: 导入过程中出现异常怎么办？
A: 系统会自动回滚事务，不会导入部分数据。请检查文件格式和数据后重试。

### Q: 支持一次导入多少条数据？
A: 建议单次导入不超过 10000 条，数据量较大时建议分批导入。
