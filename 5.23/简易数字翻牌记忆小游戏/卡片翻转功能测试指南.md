# 卡片翻转功能 - 快速测试指南

## 🎯 功能说明

卡片翻转功能已实现，点击卡片可翻开查看数字，具有以下特性：

✅ **3D 翻转动画**：使用 Y 轴旋转实现真实的翻牌效果
✅ **平滑曲线**：Mathf.SmoothStep 使动画更自然
✅ **测试模式**：无需 GameManager 即可独立测试
✅ **状态管理**：自动追踪翻转和匹配状态

## 🚀 快速测试步骤

### 方法一：使用测试脚本（推荐）

1. **在 Unity 中创建测试场景**
   - File → New Scene
   - 保存为 "CardTestScene"

2. **创建 UI Canvas**
   - 右键 Hierarchy → UI → Canvas
   - Canvas 设置保持默认即可

3. **创建测试区域**
   - 在 Canvas 下创建 Empty GameObject，命名为 "TestArea"
   - 添加 Grid Layout Group 组件：
     - Cell Size: X = 120, Y = 160
     - Spacing: X = 20, Y = 20
     - Constraint: Fixed Column Count = 4

4. **创建测试管理器**
   - 在 Hierarchy 创建 Empty，命名为 "CardFlipTester"
   - 添加 `CardFlipTester.cs` 脚本

5. **配置引用**
   - 将卡片预制体拖拽到 "Card Prefab" 字段
   - 将 TestArea 拖拽到 "Test Area" 字段
   - 设置 Test Card Count = 8

6. **运行测试**
   - 点击 Play 按钮
   - 点击任意卡片，观察翻转效果！

### 方法二：单卡片测试

1. **创建测试卡片**
   - 将卡片预制体拖入场景
   - 选中卡片，在 Inspector 中找到 Card 组件
   - 勾选 **Test Mode** 选项

2. **设置数字显示**
   - 在 Card 组件上右键 → Test
   - 或直接在代码中调用 `card.SetCardData(1, null, 5)`

3. **运行游戏**
   - 点击卡片即可翻转查看数字

## 🔧 卡片预制体检查清单

确保你的卡片预制体包含以下组件：

```
Card (GameObject)
├── Image (背景)
├── Card.cs 脚本
│   ├── Front Image  → 引用子对象 FrontImage
│   ├── Back Image   → 引用子对象 BackImage
│   ├── Card Number Text → 引用子对象 CardNumberText
│   └── Test Mode    ☐ (测试时勾选)
├── FrontImage (子 Image)
│   └── 初始状态: Set Active = false
├── BackImage (子 Image)
│   └── 初始状态: Set Active = true
└── CardNumberText (子 Text)
    └── 初始状态: Set Active = false
```

## 🎨 动画调整参数

在 Card.cs  Inspector 中可调整：

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| Flip Duration | 翻转动画时长 | 0.3 - 0.5 秒 |
| Use Smooth Curve | 使用平滑曲线 | 勾选 ✓ |
| Test Mode | 测试模式（独立运行）| 测试时勾选 |

## 📝 核心 API

```csharp
// 翻转卡片
card.FlipCard();

// 设置卡片数据
card.SetCardData(cardID, frontSprite, displayNumber);

// 检查状态
bool isFlipped = card.isFlipped;
bool isMatched = card.isMatched;

// 重置卡片
card.ResetCard();

// 标记为匹配
card.SetMatched();
```

## 🐛 常见问题

**Q: 点击卡片没反应？**
- 检查是否有 Button 组件或 Ensure Canvas Group 存在
- 确认 Test Mode 已勾选（独立测试时）
- 检查 Canvas 是否正确设置

**Q: 翻转后看不到数字？**
- 确认 CardNumberText 引用正确
- 检查 Text 颜色是否与背景冲突
- 确认 Text 的字体大小合适

**Q: 动画卡顿？**
- 尝试减少 Flip Duration 到 0.3 秒
- 确保没有其他协程阻塞

## ✅ 验证翻转功能

运行测试后，确认以下行为：

1. ▢ 点击卡片 → 卡片沿 Y 轴翻转
2. ▢ 翻转完成后 → 显示正面图片和数字
3. ▢ 再次点击 → 卡片翻回背面
4. ▢ 动画流畅 → 无卡顿或闪烁
5. ▢ 多卡片测试 → 每张卡片独立工作
