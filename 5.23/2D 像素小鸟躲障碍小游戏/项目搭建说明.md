# Unity2D 像素小鸟躲障碍小游戏 - 项目搭建说明

## 项目结构

```
2D 像素小鸟躲障碍小游戏/
├── Assets/
│   ├── Scripts/           # C# 脚本文件
│   │   ├── BirdController.cs      # 小鸟控制器
│   │   ├── Pipe.cs                # 管道脚本
│   │   ├── PipeSpawner.cs         # 管道生成器
│   │   └── GameManager.cs         # 游戏管理器
│   ├── Prefabs/           # 预制体文件夹
│   ├── Scenes/            # 场景文件夹
│   ├── Sprites/           # 精灵图片文件夹
│   └── Materials/         # 材质文件夹
├── Packages/              # 包管理
├── ProjectSettings/       # 项目设置
└── 项目搭建说明.md
```

## Unity 项目搭建步骤

### 1. 打开 Unity 项目
- 启动 Unity Hub
- 点击 "Open" 或 "Add"
- 选择本项目文件夹

### 2. 创建游戏场景
1. 在 `Assets/Scenes` 文件夹中右键 → Create → Scene
2. 命名为 "GameScene"
3. 双击打开场景

### 3. 设置摄像机
1. 选中 Main Camera
2. 设置 Projection 为 Orthographic
3. 设置 Size 为 5
4. 设置 Background 为天空蓝色（如 #87CEEB）

### 4. 创建小鸟角色
1. 在 Hierarchy 中右键 → Create Empty，命名为 "Bird"
2. 添加组件：
   - Sprite Renderer（添加小鸟精灵图片）
   - Rigidbody2D（Gravity Scale = 3）
   - Circle Collider2D（Radius = 0.4）
   - BirdController 脚本
3. 将 Bird 拖入 `Assets/Prefabs` 文件夹创建预制体

### 5. 创建管道预制体
1. 创建空对象，命名为 "Pipe"
2. 创建两个子对象：
   - "PipeTop"（上方管道）：
     - 添加 Sprite Renderer
     - 添加 Box Collider2D
     - 位置：Y = 2
   - "PipeBottom"（下方管道）：
     - 添加 Sprite Renderer
     - 添加 Box Collider2D
     - 位置：Y = -2
3. 创建 "ScoreZone" 子对象：
   - 添加 Box Collider2D（Is Trigger = true）
   - 设置 Tag 为 "ScoreZone"
   - 位置：X = 0, Y = 0
   - 大小：宽度覆盖管道间隙
4. 添加 Pipe 脚本到 Pipe 根对象
5. 将 Pipe 拖入 `Assets/Prefabs` 文件夹创建预制体

### 6. 创建管道生成器
1. 创建空对象，命名为 "PipeSpawner"
2. 添加 PipeSpawner 脚本
3. 在 Inspector 中：
   - Pipe Prefab：拖入 Pipe 预制体
   - Spawn Rate：2
   - Min Y：-2
   - Max Y：2
   - Spawn X Position：10

### 7. 创建游戏管理器
1. 创建空对象，命名为 "GameManager"
2. 添加 GameManager 脚本
3. 创建 UI Canvas：
   - 右键 → UI → Canvas
   - 创建分数显示 Text，命名为 "ScoreText"
   - 创建开始面板和游戏结束面板
4. 在 GameManager Inspector 中关联引用

### 8. 添加地面（可选）
1. 创建 Sprite，命名为 "Ground"
2. 添加 Box Collider2D
3. 放置在屏幕底部

### 9. 保存场景
- File → Save Scene
- 保存到 `Assets/Scenes/GameScene.unity`

## 控制说明

- **空格键** 或 **鼠标左键**：小鸟跳跃
- 穿过管道得分
- 碰撞管道或地面游戏结束

## 脚本功能说明

### BirdController.cs
- 处理跳跃输入
- 控制小鸟旋转
- 碰撞检测
- 计分触发

### Pipe.cs
- 管道向左移动
- 自动销毁超出屏幕的管道

### PipeSpawner.cs
- 定时生成管道
- 随机高度
- 清除所有管道功能

### GameManager.cs
- 单例模式
- 游戏状态管理
- 计分系统
- UI 面板控制
- 最高分记录
