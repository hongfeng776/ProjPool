# 管道预制体设置指南

## 管道预制体结构

```
Pipe (父对象)
├── PipeTop (上管道)
├── PipeBottom (下管道)
└── ScoreTrigger (计分触发器)
```

## 详细设置步骤

### 1. 创建管道父对象

1. 在 Hierarchy 中右键 → Create Empty
2. 命名为 "Pipe"
3. 添加 `Pipe` 脚本组件

### 2. 创建上管道 (PipeTop)

1. 在 Pipe 下创建子对象，命名为 "PipeTop"
2. 添加组件：
   - **Sprite Renderer**: 设置管道精灵图片
   - **Box Collider 2D**: 
     - Size: 根据精灵大小调整 (例如: X=1, Y=10)
     - Offset: 调整到管道中心位置
3. 位置设置：
   - Position: Y = 3.5 (使管道间隙中心为 0)

### 3. 创建下管道 (PipeBottom)

1. 在 Pipe 下创建子对象，命名为 "PipeBottom"
2. 添加组件：
   - **Sprite Renderer**: 设置管道精灵图片
   - **Box Collider 2D**: 
     - Size: 根据精灵大小调整 (例如: X=1, Y=10)
     - Offset: 调整到管道中心位置
3. 位置设置：
   - Position: Y = -3.5
   - Rotation: Z = 180 (翻转管道)

### 4. 创建计分触发器 (ScoreTrigger)

1. 在 Pipe 下创建子对象，命名为 "ScoreTrigger"
2. 添加组件：
   - **Box Collider 2D**: 
     - 勾选 `Is Trigger`
     - Size: X=0.5, Y=3 (覆盖管道间隙)
     - Offset: X=0, Y=0
3. 添加 `ScoreTrigger` 脚本组件
4. 位置设置：Position = (0, 0, 0)

### 5. 标签设置

1. 选择小鸟对象
2. 在 Inspector 顶部的 Tag 下拉菜单中
3. 选择 "Player" 标签（如果没有则创建）

### 6. 创建预制体

1. 将 Hierarchy 中的 "Pipe" 对象拖入 `Assets/Prefabs` 文件夹
2. 删除 Hierarchy 中的 Pipe 对象

### 7. 设置管道生成器

1. 在场景中创建空对象，命名为 "PipeSpawner"
2. 添加 `PipeSpawner` 脚本组件
3. 在 Inspector 中设置：
   - **Pipe Prefab**: 拖入刚才创建的 Pipe 预制体
   - **Spawn Rate**: 2 (管道生成间隔，秒)
   - **Min Y**: -2.5 (管道最低位置)
   - **Max Y**: 2.5 (管道最高位置)
   - **Spawn X Position**: 12 (管道生成的 X 坐标)

## 难度系统说明

PipeSpawner 包含自动难度递增系统：

- 每 10 秒增加一次难度
- 管道生成间隔减少 0.1 秒（最小 0.8 秒）
- 管道间隙减少 0.1 单位（最小 2 单位）

可在 Inspector 中调整这些参数。

## 计分系统说明

1. 小鸟穿过 ScoreTrigger 时触发计分
2. ScoreTrigger 会检查小鸟的 "Player" 标签
3. 每个管道只计一次分（hasScored 标志）
4. 分数通过 GameManager 更新并显示

## 调试提示

- 如果管道不移动：检查 GameManager 的 IsPlaying 状态
- 如果不计分：检查小鸟的 Tag 是否为 "Player"，检查 ScoreTrigger 的 IsTrigger 是否勾选
- 如果管道不生成：检查 PipeSpawner 的 Pipe Prefab 是否赋值
