# 个人运动步数分析可视化工具 - 快速开始

## 🚀 一键运行

### Windows 用户
双击运行 `run.bat` 即可自动完成所有步骤！

### 通用方式
在命令行执行：
```bash
python main.py
```

---

## 📋 功能说明

### 自动数据生成
- ✅ 首次运行自动生成 60 天示例数据
- ✅ 无需手动准备数据文件即可体验

### 数据导入支持
- 📄 CSV 文件 (.csv)
- 📊 Excel 文件 (.xlsx, .xls)
- 📋 JSON 文件 (.json)

### 自动校验功能
- 🔍 空值检测与处理
- ❌ 异常值过滤
- 📅 日期格式验证
- 🔢 步数范围校验
- 📊 数据质量报告

---

## 📁 自定义数据

将你的数据文件放入 `data/` 目录，支持以下列名：

| 日期列名 | 步数列名 |
|---------|---------|
| date, 日期, 时间 | steps, 步数, step_count |
| datestr, 记录日期 | 步行数, 今日步数 |

---

## 📊 输出内容

运行完成后，`output/` 目录将包含：

### 图表文件
- `daily_steps_trend.png` - 每日步数趋势
- `weekly_analysis.png` - 每周统计分析
- `weekly_average_bar.png` - 周均步数柱状图（标注极值日期）
- `weekly_extremes.png` - 每周最高/最低对比
- `monthly_analysis.png` - 每月统计分析
- `steps_distribution.png` - 步数分布图

### 统计结果
- `daily_average.csv` - 各周日均统计
- `weekly_average.csv` - 每周统计数据
- `monthly_average.csv` - 每月统计数据

---

## ⚙️ 配置说明

创建 `config.json` 可自定义配置：

```json
{
    "data_file": "data/steps_data.csv",
    "output_dir": "output",
    "min_steps": 0,
    "max_steps": 100000,
    "target_steps": 10000
}
```
