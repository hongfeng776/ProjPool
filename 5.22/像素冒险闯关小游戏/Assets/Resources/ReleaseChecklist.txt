游戏发布检查清单
==================

## 一、项目设置

### 1.1 Player Settings
- [ ] Company Name: 填写公司/团队名称
- [ ] Product Name: 游戏名称
- [ ] Version: 版本号 (如 1.0.0)
- [ ] Bundle Identifier: 包名 (如 com.company.gamename)
- [ ] Default Orientation: Landscape Left/Right
- [ ] Target Resolution: 1920x1080 或 1280x720

### 1.2 Quality Settings
- [ ] 设置各画质等级参数
- [ ] 禁用不需要的效果
- [ ] 移动端使用较低画质

### 1.3 Graphics Settings
- [ ] 配置管线设置
- [ ] 设置默认材质
- [ ] 预加载着色器

### 1.4 Input Manager
- [ ] 确认所有输入轴已配置
- [ ] Horizontal / Vertical 轴
- [ ] Jump 按钮
- [ ] Submit / Cancel 按钮

### 2.1 Build Settings
- [ ] Boot (索引 0)
- [ ] MainMenu
- [ ] LevelSelect
- [ ] GameScene
- [ ] 其他关卡场景

### 2.2 场景内容检查

- [ ] 光照设置正确

### 3.2 玩家控制

- [ ] 边界限制有效
- [ ] 不穿墙/不卡墙

### 3.3 道具系统

### 3.4 关卡系统
- [ ] LevelManager 正常加载

- [ ] 进度存档正常
- [ ] 关卡解锁正确

### 3.5 UI 系统
- [ ] 主菜单按钮正常
- [ ] 关卡选择正常
- [ ] 暂停菜单正常
- [ ] 游戏结束界面正常

## 四、性能检查

### 4.1 内存
- [ ] 无内存泄漏
- [ ] 场景切换后内存释放
- [ ] 常驻内存合理 (< 500MB)

### 4.2 帧率
- [ ] 目标平台稳定 60 FPS
- [ ] 无明显卡顿
- [ ] GC 分配合理

### 4.3 加载
- [ ] 场景加载时间 < 3秒
- [ ] 加载屏幕正常显示
- [ ] 过渡动画流畅

- [ ] 游戏失败 → 重试 正常
- [ ] 游戏胜利 → 下一关 正常
- [ ] 返回主菜单 正常

### 5.3 难度系统
