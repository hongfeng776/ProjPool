# 材料力学仿真分析工具

一个基于PyQt5和VTK的材料力学仿真分析工具，提供参数输入、有限元计算和3D结果可视化功能。

## 功能特性

### 1. 参数输入模块
- **材料参数**
  - 支持多种预设材料（结构钢、铝合金、不锈钢、钛合金）
  - 可自定义弹性模量、泊松比、密度、屈服强度
  - 材料选择时自动填充对应参数

- **载荷参数**
  - 支持集中力、均布压力、力矩、重力、温度载荷
  - 可设置载荷大小和方向向量

- **边界条件**
  - 固定约束、铰接约束、滑动约束、对称约束
  - 各方向平动和转动自由度控制

- **网格设置**
  - 四面体、六面体、混合网格
  - 可设置单元尺寸和网格密度

### 2. 模型导入功能
支持以下3D模型格式：
- STL (.stl)
- STEP/STP (.step, .stp)
- OBJ (.obj)
- PLY (.ply)
- VTK (.vtk)
- VTP (.vtp)

### 3. 有限元计算模块
- 基于简化弹性力学的应力应变计算
- 支持多种结果类型：
  - 冯·米塞斯应力（Von Mises Stress）
  - 节点位移
  - 第一、第二、第三主应力
  - X/Y/Z方向应力分量
- 自动计算安全系数并进行安全评估
- 多线程计算，避免界面卡顿

### 4. 结果可视化功能
- **应力云图显示**：8种结果类型可切换
- **颜色图例**：带数值标注的颜色条
- **5种颜色映射方案**：
  - 彩虹色
  - 蓝-红渐变
  - 灰度
  - 热图
  - 冷图
- **实时统计**：显示最大值、最小值、平均值、标准差
- **安全系数评估**：自动判断设计是否安全

### 5. 3D可视化功能
- VTK高性能渲染引擎
- 视角控制（旋转、平移、缩放）
- 线框/表面模式切换
- 坐标轴显示
- 模型信息统计（顶点数、单元数）

## 安装与运行

### 环境要求
- Python 3.7+
- PyQt5
- VTK
- NumPy

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python main.py
```

## 使用说明

### 快速开始

1. **导入模型**
   - 点击菜单 "文件" -> "导入模型"
   - 选择支持的3D模型文件
   - 模型将在右侧视图区显示
   - 如无模型，可直接使用默认示例模型

2. **设置参数**
   - 在左侧参数面板中设置材料参数
   - 设置载荷类型、大小和方向
   - 配置边界条件
   - 设置网格参数

3. **开始计算**
   - 点击"开始计算"按钮
   - 等待计算完成（进度条显示中）
   - 计算完成后自动显示应力云图

4. **查看结果**
   - 在结果显示下拉框中切换不同的结果类型
   - 切换颜色映射方案
   - 查看底部的结果统计信息
   - 左侧面板显示详细的计算结果和安全评估

### 3D视图操作

- **鼠标左键拖动**：旋转视角
- **鼠标右键拖动**：平移视角
- **鼠标滚轮**：缩放视图
- **工具栏按钮**：
  - 重置视角：恢复默认视角
  - 适应窗口：自动缩放以完整显示模型
  - 线框模式：切换线框/表面显示
  - 显示坐标轴：切换坐标系显示

### 结果类型说明

1. **冯·米塞斯应力**：综合等效应力，用于判断材料屈服
2. **位移**：节点变形量
3. **第一主应力**：最大拉应力
4. **第二主应力**：中间主应力
5. **第三主应力**：最大压应力
6. **X/Y/Z方向应力**：各方向应力分量

## 使用提示

1. **安全系数评估**
   - 安全系数 < 1.0：存在破坏风险，需要优化设计
   - 1.0 ≤ 安全系数 < 1.5：安全系数较低，建议优化
   - 安全系数 ≥ 1.5：安全系数符合常规工程要求

2. **颜色映射选择**
   - 彩虹色：适合显示整体分布
   - 蓝-红：适合对比高低值
   - 热图/冷图：适合强调极值区域
   - 灰度：适合打印输出

3. **多线程计算**
   - 计算过程中界面保持响应
   - 可随时操作3D视图查看模型
